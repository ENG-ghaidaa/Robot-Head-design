#  Robot Head Design

This project is a simple 3D CAD model of a **Robot Head** designed for engineering practice using **Onshape**. It demonstrates the creation of a mechanical part along with a fully dimensioned technical drawing.

##  Features

- 3D CAD model created in Onshape
- Front, Side, and Top views
- Isometric view
- Fully dimensioned engineering drawing
- A4 technical drawing layout
- Simple geometric features for CAD practice

## 🛠 Software Used

- Onshape

## 📐 Dimensions

| Feature | Size |
|---------|------|
| Width | 70 mm |
| Height | 50 mm |
| Top Cylinder Height | 30 mm |
| Left Hole Diameter | Ø15 mm |
| Right Hole Diameter | Ø25.47 mm |
| Inner Circle Diameter | Ø8.25 mm |
